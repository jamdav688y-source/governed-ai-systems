Overview

This repository exists to make AI governance operational.

Most teams already have policies.

Many have monitoring.

Some have evaluations.

Very few have a control surface that answers:

- who owns the decision
- who can stop it
- whether rollback is real
- whether evidence survives failure

This repository is organized around that boundary.

Use it to:

- define doctrine
- audit workflows
- package diagnostics
- enforce governance checks in CI
